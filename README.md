# Bedaya - منصة إدارة الأعمال المتكاملة

نظام متكامل لإدارة المبيعات والحجوزات والتواصل مع العملاء، مع لوحة تحكم احترافية.

## المميزات

- ✅ لوحة تحكم حديثة مع إحصائيات فورية
- ✅ نظام مبيعات متكامل
- ✅ إدارة الحجوزات
- ✅ نظام تواصل مع العملاء
- ✅ رفع وإدارة ملفات النتائج
- ✅ إرسال البريد الإلكتروني
- ✅ مصادقة JWT
- ✅ تصميم RTL عربي
- ✅ وضع مظلم/فاتح
- ✅ رسوم بيانية للمبيعات
- ✅ بحث وتصفية متقدم
- ✅ تصميم متجاوب بالكامل

## المتطلبات الأساسية

- Node.js (v14 أو أعلى)
- حساب Firebase
- حساب SMTP (Gmail, SendGrid, إلخ)
